## clone作成の際の必須機能

宿泊施設の検索機能

宿泊施設の予約機能

宿泊施設のレビュー機能

ホストとの連絡機能(with ユーザー)

## 追加機能
### ツアー機能
複数の宿泊施設への一括予約機能
### 作成した理由
一週間やそれ以上の旅行の際に一箇所に宿泊することは稀だと思い、行きたい観光地(旅の予定)を順番に入れれば泊まれる場所は提案し、一括予約ができれば便利かと思ったため
